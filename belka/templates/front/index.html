API generator

Здесь можно создать своё API. Вы создаёте схему объектов, нахуяриваете туда данных и получаете эндпоинт вида GET /api/v1/{api_name}, который отдаёт эти объекты. У эндпоинта есть Query-параметры:

  • ?filter.{field}={value} Фильтрация по значению поля {field}
  • ?search.{field}={q} Поиск по значению поля {field}
  • ?page={n} Номер страницы. Default=0
  • ?page_size={n} Размер страницы. Default=20
